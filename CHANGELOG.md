# Changelog

All notable changes to Keyfount Desktop are documented here. The version of
each release is driven by the `release:*` label on its `develop → main` pull
request; the git tag `vX.Y.Z` is the source of truth.

## v0.0.1 — 2026-05-28

Initial release of the Keyfount desktop app, shipped as a macOS Universal DMG
(Apple Silicon + Intel). This build is not yet code-signed or notarised.
